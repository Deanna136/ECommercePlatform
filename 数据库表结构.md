## 二、表结构详情

### 2.1 administrator 管理员表

> 系统唯一超级管理员，由开发者初始化，不对外开放注册。

```sql
CREATE TABLE administrator (
    id          BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '管理员id',
    user_name   VARCHAR(32) NOT NULL DEFAULT 'admin' COMMENT '用户名',
    password    VARCHAR(32) NOT NULL DEFAULT '123456' COMMENT '密码',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '最后修改时间',
    CONSTRAINT user_name UNIQUE (user_name),
    CHECK (CHAR_LENGTH(password) >= 6)
) COMMENT '管理员表';
```

**字段说明：**

| 字段名 | 数据类型 | 允许空 | 默认值 | 说明 |
|--------|----------|--------|--------|------|
| id | BIGINT | ❌ | 自增 | 主键，管理员ID |
| user_name | VARCHAR(32) | ❌ | `admin` | 用户名，全表唯一 |
| password | VARCHAR(32) | ❌ | `123456` | 密码，长度不少于6位 |
| create_time | DATETIME | ✅ | 当前时间 | 记录创建时间 |
| update_time | DATETIME | ✅ | 当前时间 | 记录最后修改时间，自动更新 |

**约束：**

| 约束类型 | 字段 | 说明 |
|---------|------|------|
| PRIMARY KEY | `id` | 主键 |
| UNIQUE | `user_name` | 用户名唯一 |
| CHECK | `password` | 密码长度 ≥ 6 |

### 2.4 product 商品表

> 存储平台所有商品信息，商品不可物理删除，通过 status 字段标记状态。

```sql
CREATE TABLE product (
    id          BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '商品id',
    product_no  VARCHAR(32) NOT NULL COMMENT '商品编号',
    name        VARCHAR(128) NOT NULL COMMENT '商品名',
    category    ENUM('clothing','food','electronics','home_living',
                     'mother_baby','sports','books','others') NOT NULL COMMENT '商品种类',
    sku         BIGINT NOT NULL COMMENT '库存',
    price       DECIMAL(10,2) NOT NULL COMMENT '价格',
    image       VARCHAR(500) COMMENT '存储商品图片的url',
    description VARCHAR(512) COMMENT '商品描述',
    seller_id   BIGINT NOT NULL COMMENT '卖家id',
    status      ENUM('pending_review','approved','onsale','suspend',
                     'offsale','rejected','out_of_stock','deleted') COMMENT '商品状态',
    sales_count INT NOT NULL DEFAULT 0 COMMENT '销量',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '最后修改时间',
    UNIQUE KEY uk_product_seller (product_no, seller_id),
    FOREIGN KEY fk_product_seller_id (seller_id) REFERENCES seller(id),
    CHECK (sku >= 0),
    CHECK (price >= 0),
    CHECK (sales_count >= 0)
) COMMENT '商品表';
```

**字段说明：**

| 字段名 | 数据类型 | 允许空 | 默认值 | 说明 |
|--------|----------|--------|--------|------|
| id | BIGINT | ❌ | 自增 | 主键，商品ID |
| product_no | VARCHAR(32) | ❌ | - | 商品编号，同一卖家下唯一 |
| name | VARCHAR(128) | ❌ | - | 商品名称 |
| category | ENUM | ❌ | - | 商品类型，见枚举值说明 |
| sku | BIGINT | ❌ | - | 库存数量，不能小于0 |
| price | DECIMAL(10,2) | ❌ | - | 商品价格（元），精确到分，不能小于0 |
| image | VARCHAR(500) | ✅ | NULL | 商品图片URL（存储于腾讯云COS） |
| description | VARCHAR(512) | ✅ | NULL | 商品描述 |
| seller_id | BIGINT | ❌ | - | 所属卖家ID，外键关联 seller.id |
| status | ENUM | ✅ | NULL | 商品状态，见枚举值说明 |
| sales_count | INT | ❌ | `0` | 累计销量，不能小于0 |
| create_time | DATETIME | ✅ | 当前时间 | 创建时间 |
| update_time | DATETIME | ✅ | 当前时间 | 最后修改时间，自动更新 |

**约束：**

| 约束类型 | 字段 | 说明 |
|---------|------|------|
| PRIMARY KEY | `id` | 主键 |
| UNIQUE KEY | `(product_no, seller_id)` | 同一卖家下商品编号唯一 |
| FOREIGN KEY | `seller_id → seller.id` | 外键，关联卖家表 |
| CHECK | `sku` | 库存 ≥ 0 |
| CHECK | `price` | 价格 ≥ 0 |
| CHECK | `sales_count` | 销量 ≥ 0 |

### 2.7 orders 订单表

> 存储订单主记录，订单不可物理删除，通过 status 字段标记状态。

```sql
CREATE TABLE orders (
    id          BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '订单id',

    seller_id   BIGINT NOT NULL COMMENT '卖家id',
    amount      DECIMAL(10,2) COMMENT '总金额',
    status      ENUM('pending','paid','processing','shipped','delivered',

    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '最后修改时间',
    FOREIGN KEY fk_order_buyer_id (buyer_id) REFERENCES buyer(id),
    FOREIGN KEY fk_order_seller_id (seller_id) REFERENCES seller(id)

### 2.8 order_items 订单商品明细表

> 存储订单中每条商品记录，包含下单时的价格快照，不随后续商品改价变化。

```sql
CREATE TABLE order_items (
    id          BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '主键',
    order_id    BIGINT NOT NULL COMMENT '订单id',
    product_id  BIGINT NOT NULL COMMENT '商品id',
    quantity    INT NOT NULL COMMENT '商品数量',
    unit_price  DECIMAL(10,2) NOT NULL COMMENT '商品单价',
    total_price DECIMAL(10,2) NOT NULL COMMENT '商品总价',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '最后修改时间',
    FOREIGN KEY fk_order_items_product_id (product_id) REFERENCES product(id),
    FOREIGN KEY fk_order_items_orders_id (order_id) REFERENCES orders(id),
    CHECK (quantity > 0),
    CHECK (unit_price >= 0),
    CHECK (total_price >= 0)
) COMMENT '订单商品关联表';
```

**字段说明：**

| 字段名 | 数据类型 | 允许空 | 默认值 | 说明 |
|--------|----------|--------|--------|------|
| id | BIGINT | ❌ | 自增 | 主键 |
| order_id | BIGINT | ❌ | - | 所属订单ID，外键关联 orders.id |
| product_id | BIGINT | ❌ | - | 商品ID，外键关联 product.id |
| quantity | INT | ❌ | - | 购买数量，必须大于0 |
| unit_price | DECIMAL(10,2) | ❌ | - | 下单时商品单价快照，不随改价变化，≥ 0 |
| total_price | DECIMAL(10,2) | ❌ | - | 该商品小计金额（= unit_price × quantity），≥ 0 |
| create_time | DATETIME | ✅ | 当前时间 | 创建时间 |
| update_time | DATETIME | ✅ | 当前时间 | 最后修改时间，自动更新 |

**约束：**

| 约束类型 | 字段 | 说明 |
|---------|------|------|
| PRIMARY KEY | `id` | 主键 |
| FOREIGN KEY | `order_id → orders.id` | 外键，关联订单表 |
| FOREIGN KEY | `product_id → product.id` | 外键，关联商品表 |
| CHECK | `quantity` | 数量必须 > 0 |
| CHECK | `unit_price` | 单价必须 ≥ 0 |
| CHECK | `total_price` | 总价必须 ≥ 0 |

> **说明：** `product` 表中已有 `seller_id` 字段，本表为辅助关联表，可用于快速反查某卖家的商品列表。

## 四、表关系说明

### 关系总览图（文字版 ER）

```
administrator (1)
    └── 独立存在，无外键关联

buyer (1) ──────────────────── cart (1)
                                    └── cart_items (N) ──── product (1)

buyer (1) ──────────────────── orders (N)
                                    └── order_items (N) ──── product (1)

seller (1) ─────────────────── orders (N)
seller (1) ─────────────────── product (N)
                                    └── seller_products (N，辅助关联)
```

### 详细关系说明

| 关系 | 类型 | 说明 |
|------|------|------|
| `buyer` → `cart` | 1 对 1 | 每个买家拥有唯一一个购物车 |
| `cart` → `cart_items` | 1 对 N | 购物车包含多条商品记录 |
| `cart_items` → `product` | N 对 1 | 多条购物车记录对应同一商品 |
| `buyer` → `orders` | 1 对 N | 买家可有多笔订单 |
| `seller` → `orders` | 1 对 N | 卖家可收到多笔订单 |
| `orders` → `order_items` | 1 对 N | 一笔订单包含多条商品明细 |
| `order_items` → `product` | N 对 1 | 多条订单明细对应同一商品 |
| `seller` → `product` | 1 对 N | 卖家可发布多个商品 |
| `seller` → `seller_products` | 1 对 N | 辅助关联，卖家与商品关系 |
| `product` → `seller_products` | 1 对 N | 辅助关联，商品与卖家关系 |
